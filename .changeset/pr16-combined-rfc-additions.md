Add new standards-focused utilities across HTTP, OAuth, JSON, and WebAuthn: RFC 6585 additional status codes, RFC 6902 JSON Patch, RFC 7396 JSON Merge Patch, RFC 7636 PKCE, RFC 8297 Early Hints Link handling, RFC 8414 OAuth authorization server metadata, RFC 8785 JSON Canonicalization Scheme, W3C Clear-Site-Data, W3C CSP3 server-side subset, W3C Referrer-Policy, W3C Reporting API helpers, and WebAuthn v1 auth parsing/verification primitives.
